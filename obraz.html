Uwagi na temat JS:
  • kiedy powstała:

    Język JavaScript powstał w 1995 roku.

  • dlaczego jest to język skryptowy:

    Nie musi zostać skompilowany do kodu maszynowego,
    aby można było zobaczyć efekty jego działania.

  • gdzie jest wykonywany JS (klient lub serwer):

    Kod JavaScript może być umieszczony w osobnych plikach,
    do których odwołują się pliki HTML, lub bezpośrednio
    w kodzie HTML w specjalnych znacznikach skryptów.

  • dlaczego jest to bezpieczne narzędzie:

    JavaScript jest nieustannie rozwijany.

  • czy jest to język obiektowy, jeśli tak to dlaczego:

    JavaScript jest językiem obiektowym skryptowym. Oznacza to,
    że jest bardzo skompilowany i zaprojektowany, by działać w interakcji
    z użytkownikiem

    ‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›

    kod tworzy przycisk pod nazwą ,,przycisk", który po wciśnięciu otwiera nowe okno o wartości ,,Nowa Strona".
